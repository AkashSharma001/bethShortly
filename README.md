## BETH Shortener URL

### Project Description

This project is a simple URL shortener built using the following technologies:

* **Elysia:** A Bun-first, performance-focused web framework.
* **HTMX:** A JavaScript library for building dynamic web applications without writing backend code.
* **Turso:** An open-source Edge database, bringing the developer experience of SQLite closer to the users.
* **Bun:** A fast and JavaScript-first runtime.

**Features:**

* Shorten any URL to a custom alias.
* Redirect users to the original URL when they click the shortened link.

### Future Updates

* **Authentication:** Implement user authentication to allow users to create and manage their own shortened URLs.
* **Custom Shortener URLs:** Allow users to define their own custom shortened URLs instead of relying on the automatically generated ones.
* **API:** Develop an API to allow programmatic access to shortened URL features.

### Dependencies

* Bun
* Elysia
* HTMX
* Turso

### Installation and Usage

1. Install the dependencies:

```
bun install
```

2. Run the development server:

```
bun run dev
```

3. Open http://localhost:3000 in your browser.

4. Start shortening your URLs!
